# 🃏 UNO 游戏插件

在 QQ 群或浏览器中直接玩 UNO 的 MaiBot 插件。支持真人玩家对战、Bot 玩家补充、标准 UNO 规则。

## 快速开始

### 方式一：浏览器 Web UI（推荐）

插件默认启用了 Web 可视化界面。启动 MaiBot 后，打开浏览器访问：

```
http://127.0.0.1:15810
```

即可看到 UNO 游戏界面。多开几个浏览器标签页就能模拟多人对战。

### 方式二：QQ 群聊

在群聊中发送：

```
/uno
```

**一局游戏的典型流程：**

```
①  /uno 创建
② 其他玩家: /uno 加入 NR3A2
③ 房主: /uno 加机器人
④ 房主: /uno 开始
⑤ 查看手牌: /uno 手牌  （私聊发送）
⑥ 出牌: /uno 出 3
```

## QQ 群聊命令

| 命令 | 说明 | 使用时机 |
|---|---|---|
| `/uno` 或 `/uno 帮助` | 显示命令列表 | 任何时候 |
| `/uno 创建` | 新开一个房间，你是房主 | 未加入房间时 |
| `/uno 加入 <房间号>` | 加入别人的房间 | 未加入房间时 |
| `/uno 房间` | 查看所有可加入的房间 | 任何时候 |
| `/uno 开始` | 开始游戏（至少2人） | 你是房主、游戏未开始时 |
| `/uno 加机器人` | 添加一个 Bot 玩家 | 你是房主、游戏未开始时 |
| `/uno 手牌` | 查看自己的手牌（**私聊**发送） | 游戏中 |
| `/uno 出 <序号>` | 出牌，序号从 1 开始 | 轮到你的回合时 |
| `/uno 出 <序号> <颜色>` | 出万能牌时指定颜色（红/黄/蓝/绿） | 出万能牌时 |
| `/uno 摸牌` | 摸一张牌并跳过回合 | 轮到你的回合时 |
| `/uno 摸牌` | 被 +2/+4 时自动累积，再发一次摸牌 | 被罚牌时 |
| `/uno UNO` | 喊 UNO（手牌剩1张时） | 出倒数第二张牌后 |
| `/uno 抓UNO <对方QQ>` | 抓没喊 UNO 的人，他罚2张 | 别人出完倒数第二张牌但没喊时 |
| `/uno 状态` | 查看当前游戏进度 | 游戏中 |
| `/uno 退出` | 退出当前房间 | 任何时候 |

## 游戏规则

### 基本规则
- 每人起手 7 张牌
- 按顺时针轮流出牌，出的牌必须和牌面的**颜色**或**数字/符号**相同
- 没牌出时必须摸一张，如果摸的牌能出则立刻出，否则过
- 先出完所有手牌的人获胜

### 功能牌

| 牌 | 效果 |
|---|---|
| 🚫 **跳过** | 下家被跳过 |
| 🔄 **反转** | 出牌方向反转（2人局相当于跳过） |
| +2 | 下家摸 2 张并跳过回合（可叠加，下家可继续出 +2 转给再下家） |
| 🌟 **万能** | 出牌者可指定任意颜色 |
| 🌟 **+4** | 出牌者指定颜色，下家摸 4 张并跳过（可叠在 +2 或其他 +4 上） |

### UNO 喊牌
- 出到剩 1 张手牌时必须喊 "UNO"
- 没喊被抓住的话，罚摸 2 张

### Bot 玩家
- 机器人会自动出牌，等待时间约 1.5 秒
- 优先出功能牌，同色牌优先

## Web 可视化界面

插件启动后访问 `http://127.0.0.1:15810` 即可使用图形化界面：

- **大厅** - 看到所有房间列表，创建或加入房间
- **游戏界面** - 可视化显示弃牌堆、所有玩家的手牌数量和状态
- **点选出牌** - 可出的牌会有绿色边框，直接点击出牌
- **万能牌** - 出万能牌时会弹出颜色选择器
- **Bot 自动** - Bot 玩家自动出牌，界面会实时刷新
- **多人同机** - 多开浏览器标签页或窗口，每个标签页作为一个玩家

> 无需内网穿透，无需服务器，开个浏览器就能玩。

## 配置文件

`config.toml` 可选调整：

```toml
[game]
max_players = 6              # 最大玩家数
allow_public_rooms = true    # 允许公开房间
auto_cleanup_minutes = 30    # 结束30分钟后自动清理房间
bot_turn_delay_seconds = 1.5 # Bot出牌延迟

[web]
enabled = true               # Web可视化界面开关
host = "127.0.0.1"           # 监听地址（开放局域网时改为 0.0.0.0）
port = 15810                 # 监听端口
access_token = ""            # 访问令牌（留空不校验；设置后需在 URL 加 ?token=xxx）

[api]
enabled = false              # TCP API 服务开关（供 EXE/APK 客户端）
host = "127.0.0.1"
port = 15811
access_token = ""            # API 访问令牌（留空不校验）
```

> ℹ️ **隐私说明**：对局结束后，`llm.generate` 能力会将玩家昵称和胜者名发送给 LLM 生成对局解说，可在 `config.toml` 的 `[game_summary]` 中关闭（`enabled = false`）。

修改配置后执行 `/pm plugin reload uno-game-plugin` 热重载生效。

## 常见问题

**Q: 浏览器打开 http://127.0.0.1:15810 显示空白？**
A: 检查 MaiBot 是否正在运行，以及插件是否成功加载（查看日志有无报错）。

**Q: 私聊收不到手牌？**
A: Bot 账号需要和被通知的 QQ 好友关系才能发送私聊。建议使用 Web 界面游玩。

**Q: 可以中途退出吗？**
A: 可以，退出后下家接手你的回合。

**Q: 怎么让 Bot 出牌快一点？**
A: 改 `config.toml` 中的 `bot_turn_delay_seconds` 为更小值。

**Q: 能让局域网其他人一起玩吗？**
A: 将 `config.toml` 中 `host` 改为 `0.0.0.0`，其他人用你的局域网 IP 访问即可（如 `http://192.168.x.x:15810`）。

> 🚨 **安全警告**：
> Web/API 服务默认无鉴权，任何能访问该端口的人均可创建、加入、操控房间并查看其他玩家手牌。
> 在开放网络前请务必执行以下操作：
> - **设置访问令牌**：在 `config.toml` 的 `[web]` 配置中添加 `access_token = "你的随机密码"`。设置后访问所有 API 端点需携带 `?token=你的随机密码` 参数。
> - **TCP API 同理**：在 `[api]` 配置中设置 `access_token`，客户端请求时需要在 JSON 中传入 `token` 字段。
> - **不要将服务端口暴露到公网**。仅在受信任的局域网内开放。
> - **使用完毕后改回 `host = "127.0.0.1"`**。
